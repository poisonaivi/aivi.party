"brand" assets

Recommendations:

  • Scale these assets to the required size before using.
  • When using just the logo, aivi-logo-2026-pix-pad is preferable for most cases.
  • When using the logo/wordmark combination, aiviparty-horizontal is preferable for most cases.
  • Recolor these assets as needed, but if so, maintain a single, solid color.
